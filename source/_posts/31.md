## 结合性和左、右递归

```g4
grammar Expr;

expr: expr '^' expr
| INT
;

INT: [0-9]+;
WS: [ \t\r\n]+ -> skip;
```

![antlr4_parse_tree](https://user-images.githubusercontent.com/20685961/212845172-a01a0d63-a53f-4c0b-976c-586ca5d887cc.png)

使用 `assoc`
```g4
grammar Expr;

expr: <assoc=right> expr '^' expr
| INT
;

INT: [0-9]+;
WS: [ \t\r\n]+ -> skip;

```
