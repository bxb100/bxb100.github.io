## 结合性和左、右递归

```g4
grammar Expr;

expr: expr '^' expr
| INT
;

INT: [0-9]+;
WS: [ \t\r\n]+ -> skip;
```

使用 `assoc`
```g4
grammar Expr;

expr: <assoc=right> expr '^' expr
| INT
;

INT: [0-9]+;
WS: [ \t\r\n]+ -> skip;

```

### 左递归
`*( *a) [] []`

```antlr
grammar PC;

decl: decl '[' ']'
| '*' decl
| '(' decl ')'
| ID
;

ID: [a-zA-Z]+;
WS: [ \t\r\n]+ -> skip;

```
